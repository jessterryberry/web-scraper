## About The Project

[![Python Web Scraper - Niagara Construction Association][product-screenshot]](https://github.com/jessterryberry/web-scraper/blob/main/webscraper.png)

This is a python web scraper built to grab the data of the members from the Niagara Construction Association. The data was copied into an Excel file and saved as a table.

### Built With

* [![Python][Python.com]][Python-url]
* [![Jupyter][Jupyter.com]][Jupyter-url]
* [BeautifulSoup][BeautifulSoup-url]
